## Voxelization Based Global illumination （VXGI）

![image.png](assets/clip-map.png)

Clip map 更新

使用类似 Virtual texture 的方式更新数据，就是当前数据只是整块大区域中的一小块的引用

Voxelization for Opacity

Voxel 具有透明度

1. 使用 RSM 为 Voxel 注入光照信息
2. Voxel 向周围 Voxel 散射光照信息
