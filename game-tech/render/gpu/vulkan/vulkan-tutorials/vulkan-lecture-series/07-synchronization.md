### Memory Barriers

Memory Availability and Visibility ,当内存非 Availability 时，禁止后续 cmd 访问。

Excution Barriers 是从渲染流程着手，Memory Barrier 是从内存是否是最新着手，防止后续 cmd 访问，来使 cmd 同步。
