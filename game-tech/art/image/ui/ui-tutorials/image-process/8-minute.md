图层 ： 图层叠加出完整的图
- 相邻的图层可以通过混合模式起作用
- Ctrl + j  复制图层
- 图层控制
	- 没有内容，专门作为图层后处理控制的图层称为控制图层
	- 内容显示蒙版
	- 剪切蒙版, 与下一层特殊混合，和其他层普通混合

选区 ：所有的操作会在选区上操作

- 平移
- 抠图
- 限定区域操作
- Ctrl + T 完整选区图层
- 填充
	- 填充全透明 ：Delete 
	- 填充颜色 ： Alt/Ctrl + Delete
	- 智能填充 ： shift + F5

画笔
- 仿制图章 ：智能画笔，画采样后的东西

文字
- 字符图层
