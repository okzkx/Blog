## 图片显示出现问题

**原因**

github 的域名解析出错,

**解决思路**

直接修改 hosts, 帮助域名解析

**步骤**

1. 查找域名对应的 IP 地址

   [https://www.ipaddress.com/](https://www.ipaddress.com/)

   githubusercontent.com

2. 打开文件

   > C:\Windows\System32\drivers\etc\hosts

3. 在文件最后写入域名解析结果

   xxx.xxx.xxx.xxx .githubusercontent.com
