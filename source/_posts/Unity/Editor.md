# Editor 的调用及初始化



## Editor 界面点击调用静态方法



#### 菜单栏静态方法

```c#
[MenuItem("Tools/Init")]
static void Init() {}
```


#### Hierachy 或 Project 右键静态方法

实际上就是菜单栏的 Assets 中的方法

```c#
[MenuItem("Assets/Create/Init")]
static void Init() {}
```
