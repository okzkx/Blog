### 易用性三原则

- **【体验一致】** : 框架结构、色彩范围、操作反馈、文案描述的一致性
- **【行为可控】** : 玩家能充分掌控自己的交互行为
- **【状态可见】** : 在合适的时间给予适当的反馈，避免玩家迷茫

### **可读性**三原则

- 信息层级 : 玩家应该在第一时间获取到最重要的信息
- 可读易读 : 文案表达清晰，视觉观感舒适，能轻松阅读理解，长时间不疲劳
- 阅读流程 : 规划清晰的视线流程，提供连贯的阅读体验，避免跳跃式的布局
