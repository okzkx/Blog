创建骨骼
1. 内置骨骼
2. 内置插件 Rigify
3. 第三方插件 : AutoRigPro

骨骼模式
1. 物体模式
2. 编辑模式 , 创建和对位: Tab
3. 姿态模式 , 做动画 : Ctrl + Tab

骨骼编辑 : Tab
- 挤出 : E
- 扭转 : Ctrl + R
- 清空扭转 : Alt + R
- 扭转对齐 : Shift + N

蒙皮

1. 先选择模型, 
2. 再 Shift 选择骨骼, 
3. 再 Ctrl + P
4. 再设置父对象自动权重

依靠顶点组完成骨骼权重

刷权重
1. 先选择骨骼
2. 再 Shift 选择模型
3. 再进入权重绘制模式
4. Alt 切换 骨骼和顶点组
5. 刷权重 : F
6. 去掉权重 : Ctrl
